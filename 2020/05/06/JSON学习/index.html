1.JSON格式学习写博客

JSON是一种轻量级的数据交换格式，是JS的字符串表示法。

JSON可以是对象，数组，数字，字符串，TRUE,FALSE,NULL

JSON并列的值之间使用逗号，JSON对象使用{}括起来，对象的名称必须使用””，数字没有””，不支持8进制，但支持e/E(科学计数)，不能使用单引号表示字符串。

错误例子：{a:1,b:2}，没有引号

错误例子：{“a”:1,}，多了逗号

错误例子：{“a”:’a’}，使用单引号

2.JSON转换为JS对象

var obj = JSON.parse(‘{“a”: 1}’);
