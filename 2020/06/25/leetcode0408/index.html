面试题 04.08. 首个共同祖先

设计并实现一个算法，找出二叉树中某两个节点的第一个共同祖先。

不得将其他的节点存储在另外的数据结构中。

注意：这不一定是二叉搜索树。

算法思路如下：

  1. 如果root为空或者查找到了p或q，将其返回。
  2. 如果root并非p,q,null，用户分别查找左子树和右子树。
  3. 考虑当前节点root，如果Left和right都有值，表示一个节点在left的子树范围内，另一个节点在left的子树范围内。
  4. 如果只有Left找到了，返回left，同理如果只有right找到了，返回right。
  5. 假如Left和right都找不到（例如不为p,q的叶子节点），返回right = null（和最后一种情况合并）。

举个例子，

[3,5,1,6,2,0,8,null,null,7,4]

    3
   / \
  5   1
 / \ / \
6  2 0  8
  / \
 7   4

比如找 1和6，假如当前节点为 7，Left和right都为空，返回null给 2。因为 2的两个节点返回值均为Null,自己也返回Null. 对于节点 5,因为左边是6，右边2的返回值是null，返回left 6。因为5的返回值是6,而1的返回值是1，那么root满足左右不为空，返回root.
